Ejercicio 3

Función:
Bloque de código que realiza alguna operación.
Let:
Declara una variable de alcance local con ámbito de bloque (block scope), la cual, opcionalmente, puede ser inicializada con algún valor.
Const
Especifica que el valor de una variable es constante e indica al compilador que evite que el programador lo modifique.
Etiqueta Style:
Etiquetas usadas para introducir código CSS en línea, es decir, en el propio documento HTML.
Etiqueta Nav:
Usadas para definir el contenido que será la sección de navegación de la web.
Etiqueta Section:
Define una sección del documento.
